### 🟢 1. RECOVERY SORGENTE
* **Fork:** Repository già acquisito su account personale.
* **Asset:** Download pacchetto ZIP completato per migrazione su server.

---

### 🔑 2. AUTENTICAZIONE (SESSION ID)
* **Protocollo:** Generazione tramite [**Pair Code Engine**](https://knight-bot-paircode.onrender.com/).
* **Data:** Copiare la stringa `KnightBot!` ricevuta in chat privata WhatsApp.

---

### 🚀 3. INFRASTRUTTURA KATABUMP
* **Menu:** Accesso tramite icona (**≡**) -> **Order**.
* **Type:** Node.js (Configurazione standard).
* **Economia:** Sistema a gettoni. 10€ = 5 mesi di operatività. Controllare il rinnovo mensile.

---

### 🔐 4. CREDENZIALI TECNICHE
* **ID:** Visualizzato nel profilo sotto l'email.
* **Password:** Disponibile in **Dashboard** -> **Action** -> **See** -> **Change Password**.
* **Console:** Accesso tramite **GO Server** con i parametri sopra indicati.

---

### 📂 5. FILE MANAGEMENT (ALGORITMO ../)
1. Caricare lo ZIP nella sezione **Files**.
2. **Unarchive** tramite menu opzioni (⋮).
3. Spostare i file nella **Root** selezionando tutto e digitando `../` nel comando di spostamento (icona cartella con freccia).

---

### ⚙️ 6. CONFIGURAZIONE & BOOT
1. Aprire **`config.js`**.
2. Inserire `ownerNumber` e `sessionID` (tra le sbarrette `| |`).
3. **Save Content** -> **Console** -> **Start**.

---

### 📜 7. COMANDI E POLICY
* **Interfaccia:** Digitare `.menu` per l'elenco automazioni.
* **Copyright:** Progetto protetto. Vietata la duplicazione non autorizzata.
